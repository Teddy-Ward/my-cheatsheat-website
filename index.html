HTML Cheatsheet

Tables

Tag Name Description
<table>  </table> Table The wrapper element for all HTML tables.
<thead>  </thead> Table Head The set of rows defining the column headers in a table.
<tbody>  </tbody> Table Body The set of rows containing actual table data.
<tr>  </tr> Table Row The table row container.
<td>  </td> Table Data The table row container.
<tfoot>  </tfoot> Table Foot The set of rows defining the footer in a table.
colspan Coloumn Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span. 
                    <table>
                        <tr>
                            <td>Row 1, Column 1</td>
                            <td>Row 1, Column 2</td>
                        </tr>
                        
                        <tr>
                            <td>Row 2, Column 1</td>
                            <td>Row 2, Column 2</td>
                        </tr>
                    </table>

Forms

Tag Name Description
<form>  </form> Form Defines the form.
<select multiple name=? size=?>  </select> Scrolling Menu Size sets number of menu items visable before user needs to scroll.
<select name=?>  </select> Pulldown Menu Creates a pull down menu.
<textarea name=? cols="x" rows="y">  </textarea> Text Box Coloums set width, Rows set height.
<input type="checkbox" name=? value=?>  </input> Checkbox Creates a checkbox. Add 'checked' to pre-check.
<input type="radio" name=? value=?>  </input> Radio Button Creates a Radio button. Add 'checked' to pre-check.
<input type="submit" value=?>  </input> Submit Button Creates a Submit button. Value sets text inside button.
<input type="reset">  </input> Text Box Coloums set width, Rows set height.

CSS Cheatsheet

Sizing

Tag Name Description
box-sizing: content-box | border-box Content Box The content box is default.
height: <px | em | % | keyword> Set Height Sets height for non-inline elements.
width: <px | em | % | keyword> Set Width Sets width for non-inline elements.

Positioning

Tag Name Description
position: static | relative | absolute | sticky Position Sets element's positioning, along with top, left, bottom, right properties.
margin 0 auto Margin Centers block in containing block.
overflow: visible | hidden | scroll | auto | clip | hidden visible Overflow How to render content outside of content-box.
